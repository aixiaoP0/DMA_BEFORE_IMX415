# 理论最大帧率测试方法

> 摄像头 30fps 硬件限制 → 捕获一帧后绕过 poll/DQBUF，循环注入 → 测后端管线极限。

---

## 核心思路

适配我们架构（ISP 直接输出 NV12，无 MPP 解码器），修改 `Thread_ReadVideo`：

```
第 1 帧: 正常 poll → DQBUF → 缓存 NV12 数据 → QBUF（归还）
第 2 帧起: 跳过 poll/DQBUF → 使用缓存的 NV12 → Path A + Path C → enqueue
```

## 代码骨架

```cpp
void Thread_ReadVideo(app_state_t& state, int& fd,
    SafeQueue<FrameData>& g_SafeQueueRead, int& img_index, ...)
{
    cv::Mat bgr_frame(720, 1280, CV_8UC3);

    // ── 测速专用：NV12 缓存 ──
    uint8_t* cached_nv12 = nullptr;
    int cached_size = 0;
    bool bench_mode = false;   // true = 跳过 poll/DQBUF
    // ─────────────────────────

    while (true)
    {
        FrameData frame_temp;
        int width = 1280, height = 720;
        int nv12_size = width * height * 3 / 2;

        // ── 第 1 帧：正常采集 ──
        if (!bench_mode) {
            // poll + DQBUF（与原有代码完全一致）
            struct pollfd poll_fds[1];
            poll_fds[0].fd = fd;
            poll_fds[0].events = POLLIN;
            if (poll(poll_fds, 1, 5000) <= 0) continue;

            struct v4l2_buffer buf;
            struct v4l2_plane planes[1];
            memset(&buf, 0, sizeof(buf));
            memset(&planes, 0, sizeof(planes));
            buf.type = V4L2_BUF_TYPE_VIDEO_CAPTURE_MPLANE;
            buf.memory = V4L2_MEMORY_DMABUF;
            buf.m.planes = planes;
            buf.length = 1;
            if (ioctl(fd, VIDIOC_DQBUF, &buf) < 0) break;

            dmabuf_buffer_t* dmabuf_buffer = &state.dmabuf_buffer[buf.index];
            my_dmabuf_sync_start(my_dmabuf_get_fd(dmabuf_buffer));

            // 缓存 NV12（唯一一次 malloc+memcpy）
            cached_size = nv12_size;
            cached_nv12 = (uint8_t*)malloc(cached_size);
            memcpy(cached_nv12, my_dmabuf_get_mapped_addr(dmabuf_buffer), cached_size);

            my_dmabuf_sync_stop(my_dmabuf_get_fd(dmabuf_buffer));

            // 立即 QBUF 归还（不再需要摄像头了）
            memset(&buf, 0, sizeof(buf));
            memset(&planes, 0, sizeof(planes));
            buf.type = V4L2_BUF_TYPE_VIDEO_CAPTURE_MPLANE;
            buf.memory = V4L2_MEMORY_DMABUF;
            buf.m.planes = planes;
            buf.length = 1;
            buf.index = buffer_info.index;
            buf.m.planes[0].m.fd = state.buffer_infos[buf.index].dmabuf_fd;
            buf.m.planes[0].length = state.buffer_infos[buf.index].size;
            ioctl(fd, VIDIOC_QBUF, &buf);

            bench_mode = true;  // 切换到测速模式
            printf("已缓存第 1 帧，进入极限测速模式\n");
        }

        // ── 测速循环：使用缓存的 NV12 ──
        // Path A: NV12(CPU缓存) → RGA → BGR(画框)
        rga_buffer_handle_t src_handle = importbuffer_virtualaddr(
            cached_nv12, cached_size);
        rga_buffer_handle_t dst_handle = importbuffer_virtualaddr(
            bgr_frame.data, width * height * 3);

        rga_buffer_t src = wrapbuffer_handle(
            src_handle, width, height, RK_FORMAT_YCbCr_420_SP);
        rga_buffer_t dst = wrapbuffer_handle(
            dst_handle, width, height, RK_FORMAT_BGR_888);
        imcvtcolor(src, dst, RK_FORMAT_YCbCr_420_SP, RK_FORMAT_BGR_888);

        frame_temp.frame = bgr_frame.clone();
        frame_temp.index = img_index++;
        frame_temp.width = width;
        frame_temp.height = height;

        // Path C: NV12(CPU缓存) → RGA 缩放+转换 → 直写 NPU fd
        int target_worker = frame_temp.index % 4;
        if (gthreadpool.is_worker_busy(target_worker)) {
            frame_temp.skip_inference = true;  // 测速中也可能丢帧
        } else {
            gthreadpool.set_worker_busy(target_worker, true);
            int npu_fd = gthreadpool.get_worker_input_fd(target_worker);

            rga_buffer_handle_t src_c = importbuffer_virtualaddr(
                cached_nv12, cached_size);
            rga_buffer_handle_t dst_c = importbuffer_fd(
                npu_fd, 640 * 640 * 3);

            rga_buffer_t src_c2 = wrapbuffer_handle(
                src_c, width, height, RK_FORMAT_YCbCr_420_SP);
            rga_buffer_t dst_c2 = wrapbuffer_handle(
                dst_c, 640, 640, RK_FORMAT_RGB_888);
            imresize(src_c2, dst_c2);

            releasebuffer_handle(src_c);
            releasebuffer_handle(dst_c);
        }

        g_SafeQueueRead.enqueue(frame_temp);  // 背压：下游处理不过来时会阻塞

        releasebuffer_handle(src_handle);
        releasebuffer_handle(dst_handle);
    }

    if (cached_nv12) free(cached_nv12);
}
```

## 差异说明

| 对比项 | 错误的旧版本（单帧测试.txt） | ✅ 正确版本 |
|--------|---------------------------|-----------|
| MPP 解码 | 需要 `MppDecoder` | **不需要**（ISP 直接输出 NV12） |
| V4L2 类型 | `V4L2_BUF_TYPE_VIDEO_CAPTURE` | ✅ `VIDEO_CAPTURE_MPLANE` |
| QBUF | `buf.m.fd`（单平面）| ✅ `buf.m.planes[0].m.fd` |
| NV12 来源 | MPP 解码 MJPEG | ✅ ISP 硬件 NV12 |
| Path C | 无 | ✅ RGA 直写 NPU fd |

## 注意事项

1. **is_busy 限制真实**：测速循环中 `is_busy` 仍然生效，这反映了真实管线中 NPU 内存独占的限制。如果想测"完全不丢帧的极限"，可以注释掉 `is_busy` 检查，但那样会引入 NPU 内存竞争花屏——测得的数据没有实际参考价值。

2. **背压仍生效**：`g_SafeQueueRead.enqueue()` 在队列满时阻塞，所以测速不受"内存无限暴涨"干扰。

3. **NV12 缓存来自 CPU memcpy**：Path A 和 Path C 使用 `importbuffer_virtualaddr`（RGA 从 CPU 内存 DMA 读取），而不是真实的 `importbuffer_fd(dmabuf_fd)`。RGA 通过 virtualaddr 读取延迟略高于 fd，但这部分差异微乎其微（~μs 级）。

4. **测的是后端极限**：这个方法绕过的是采集端（poll/DQBUF），测的实质是 **RGA + NPU + 后处理 + 编码 + 推流** 的全链路吞吐上限。
